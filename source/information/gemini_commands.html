Gemini CLI Slash Commands

Core & Session Management

명령어설명
/help (또는 /?)Gemini CLI 및 사용 가능한 명령어에 대한 도움말 정보를 표시합니다.
/about버전 정보 및 진단 세부 정보를 표시합니다.
/quit (또는 /exit)대화 세션을 종료합니다. --delete 옵션을 사용하여 기록 및 임시 파일을 삭제할 수 있습니다.
/resume (또는 /chat)이전 세션을 탐색하고 다시 시작하거나 수동 체크포인트를 관리합니다. (list, save, delete 등)
/clear터미널 화면과 스크롤백을 지웁니다. (단축키: Ctrl+L)
/copy마지막 Gemini 출력을 시스템 클립보드에 복사합니다.
/stats사용 통계를 표시합니다. (session, model, tools)
/rewind기록을 뒤로 탐색하여 채팅이나 코드 변경 사항을 되돌립니다. (단축키: Double Esc)
/restore [tool_call_id]특정 도구 실행 전의 체크포인트로 복원하여 파일 변경 사항을 취소합니다.

Context & Memory Management

명령어설명
/memoryAI의 지침 컨텍스트(GEMINI.md)를 관리합니다. (add, list, refresh, show)
/init디렉토리를 분석하여 프로젝트 전용 GEMINI.md 파일을 생성합니다.
/compress토큰을 절약하기 위해 채팅 컨텍스트를 요약으로 대체합니다.

Configuration & Personalization

명령어설명
/settings전역 및 프로젝트 설정을 위한 대화형 편집기를 엽니다.
/model활성 AI 모델을 구성합니다.
/themeCLI의 시각적 테마를 변경하는 대화창을 엽니다.
/vim입력 프롬프트에 대해 Vim 스타일 탐색 및 편집을 토글합니다.
/editor지원되는 외부 편집기를 선택하는 대화창을 엽니다.
/terminal-setup다중 라인 입력을 위한 터미널 키 바인딩을 구성합니다.
/auth인증 방법을 변경하는 대화창을 엽니다.
/privacy개인정보 보호 고지를 표시하고 데이터 동의를 관리합니다.

Workspace & Extensions

명령어설명
/directory (또는 /dir)다중 디렉토리 작업 공간 지원을 관리합니다.
/tools사용 가능한 도구를 나열합니다.
/mcpModel Context Protocol 서버를 관리합니다.
/skills특수 워크플로우를 위한 에이전트 스킬을 관리합니다.
/agents로컬 및 원격 하위 에이전트를 관리합니다.
/extensionsCLI 확장을 관리합니다. (list, install, update 등)
/commands.toml 파일에 정의된 사용자 정의 슬래시 명령어를 관리합니다.

Advanced Features & Integration

명령어설명
/plan계획 모드로 전환하거나 현재 생성된 계획을 확인합니다.
/ideIDE 통합 및 컴패니언을 관리합니다.
/hooks동작 사용자 정의를 위한 수명 주기 후크를 관리합니다.
/shells (또는 /bashes)백그라운드 쉘 프로세스의 가시성을 토글합니다.
/policies활성 보안 및 도구 실행 정책을 나열합니다.
/permissions폴더 신뢰 설정을 관리합니다.
/bugGemini CLI에 대한 GitHub 이슈를 제출합니다.
/setup-github프로젝트 분류를 위한 GitHub Actions를 구성합니다.
/docs브라우저에서 문서를 엽니다.
/upgradeGemini Code Assist의 업그레이드 페이지를 엽니다.

HTML 기본 구조

초등학생도 쉽게 이해하는 웹페이지 만들기!

HTML이 뭔가요? HTML은 웹사이트를 만드는 언어예요. 마치 레고 블록을 쌓아서 집을 짓듯이, HTML을 쌓아서 웹페이지를 만드는 거예요!

HTML 문서의 기본 틀

HTML 문서는 항상 비슷한 틀을 가지고 있어요. 마치 학교 시간표처럼 정해진 자리가 있는 거예요.

<!DOCTYPE html>
<html>
  <head>
    <title>내 웹페이지 제목</title>
  </head>
  <body>
    <h1>안녕! 나의 첫 웹페이지</h1>
    <p>여기에 내용을 쓰면 화면에 보여요.</p>
  </body>
</html>

각 부분이 뭔지 알아볼까요?

1. <!DOCTYPE html>

이건 "이 문서는 HTML 문서예요!" 라고 컴퓨터에게 알려주는 표시예요.
마치 책 표지에 "이 책은 동화책이에요"라고 쓰는 것처럼요!

2. <html> ~ </html>

이건 HTML 문서 전체를 감싸는 가장 큰 상자예요.
모든 것이 이 안에 들어가요. 마치 도시 전체를 둘러싸는 큰 성벽 같은 거예요!
<html> (전체 상자)

├─
<head> (머리)
│ └─ <title> (제목)
├─
<body> (몸통)
│ └─ <h1> (큰 제목)
│ └─ <p> (본문)

3. <head> ~ </head>

이건 머리 부분이예요. 사람 머릿속에는 생각이 들어있잖아요? 여기도 비슷해요.
화면에는 안 보이지만 중요한 정보를 넣는 곳이예요.
<title> : 브라우저 탭(위쪽 창)에 보이는 제목이예요.
예를 들어 "네이버"나 "유튜브" 같은 거예요!

4. <body> ~ </body>

이건 몸통 부분이예요. 사람의 몸처럼, 화면에 실제로 보이는 모든 내용이 여기 들어가요.
  • <h1> : 큰 제목 (책의 큰 제목처럼)
  • <p> : 문단, 글 내용 (책의 본문처럼)

실제로 보면 이렇게 보여요!

위에 쓴 코드를 실행하면 브라우저 화면에 이렇게 나와요:

내 웹페이지 제목

안녕! 나의 첫 웹페이지

여기에 내용을 쓰면 화면에 보여요.

🧠 쉽게 외우는 방법!

"머리(head)에는 생각(title)을 담고,
몸(body)에는 보여줄 내용을 담는다!"

중요한 규칙 하나!

HTML은 태그(tag) 라는 걸 사용해요. 태그는 여는 태그 <h1> 와 닫는 태그 </h1> 가 한 쌍이예요.

마치 문을 열고(<>) 다시 닫는(</>) 것처럼요!
순서: 열기 → 내용 쓰기 → 닫기

<h1> 안녕하세요! </h1>

이렇게 안 하면 HTML이 "어? 이 문이 안 닫혔네?" 하고 헷갈려할 수 있어요.

정리하기

부분 역할 화면에 보여요?
<!DOCTYPE html> "이건 HTML이에요" 알림 ❌ 안 보여요
<html> 전체를 감싸는 상자 ❌ 안 보여요
<head> 머리 (생각/설정) ❌ 안 보여요
<title> 브라우저 탭 제목 ✅ 탭에만 보여요
<body> 몸통 (실제 내용) ✅ 보여요!
<h1> 큰 제목 ✅ 보여요!
<p> 문단/본문 ✅ 보여요!

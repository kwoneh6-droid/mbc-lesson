🏷️ class와 id 배우기

우리 웹사이트 친구들에게 특별한 이름을 붙여줘요!

🆔 id : 너는 딱 한 명뿐이야!

🪪

id는 이름표나 주민등록번호 같아요!
우리 반에서 '김철수'라는 이름표를 딱 한 명만 가질 수 있는 것과 같죠.

언제 쓰나요?

  • 웹페이지에서 단 하나뿐인 특별한 요소를 찾을 때 써요.
  • 한 페이지에 똑같은 id 이름을 두 번 쓰면 컴퓨터가 헷갈려해요!
<div id="my-hero">나는 주인공이야!</div>

CSS 꿀팁 CSS에서 id를 부를 때는 앞에 # (샵)을 붙여요!

#my-hero {
  color: gold;
}

👥 class : 우리 모두 친구!

👕

class는 반 티셔츠나 동아리 같아요!
축구부 친구들이 모두 같은 옷을 입을 수 있는 것처럼, 여러 명에게 똑같이 줄 수 있어요.

언제 쓰나요?

  • 여러 요소들을 한꺼번에 똑같이 꾸미고 싶을 때 써요.
  • 여러 개의 태그에 똑같은 class 이름을 마음껏 붙일 수 있어요!
<p class="blue-text">나도 파란색!</p>
<p class="blue-text">나도 파란색이야!</p>

CSS 꿀팁 CSS에서 class를 부를 때는 앞에 . (점)을 붙여요!

.blue-text {
  color: blue;
}

🆚 한눈에 비교하기

특징 🆔 id 👥 class
사용 횟수 페이지당 딱 1번 여러 번 중복 가능
비유 이름표, 주민번호 교복, 팀 유니폼
CSS 기호 # (샵) . (점)
중요도 매우 높음 (주인공) 보통 (여러 명)

💡 선생님의 비밀 팁!

1. 한 친구가 id와 class를 모두 가질 수도 있어요!
<div id="boss" class="red-box">

2. 한 친구가 여러 개의 class를 가질 수도 있어요!
<div class="box red-color big-size"> (한 칸씩 띄어서 써요)

3. id는 이름처럼, class는 별명처럼 생각하면 쉬워요!

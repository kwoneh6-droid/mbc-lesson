Gemini CLI 실행 가이드

윈도우 어디서든 Gemini와 대화를 시작하는 방법

방법 1: 마우스 우클릭 메뉴에 등록하기 (가장 편리함)

폴더를 열거나 폴더 아이콘 위에서 마우스 오른쪽 버튼을 눌러 바로 실행하는 방법입니다.

1

내 컴퓨터에서 아래 경로로 이동합니다:
D:\ai_contents_class\web\html

2

setup_gemini_context_menu.reg 파일을 찾아서 더블 클릭합니다.

3

보안 경고창이 뜨면 '예' 또는 '확인'을 계속 누릅니다. (레지스트리 변경 허용)

4

이제 아무 폴더나 우클릭해 보세요! 메뉴에 "Open Gemini CLI Here"가 생겼을 겁니다.

방법 2: 터미널에서 직접 실행하기

이미 시스템 설정이 되어 있어, 언제든 명령어로 저를 부를 수 있습니다.

1

키보드의 윈도우 키 + R을 누릅니다.

2

입력창에 cmd 라고 치고 엔터를 누릅니다.

3

검은 창(터미널)이 뜨면 아래 명령어를 입력합니다.

gemini

꿀팁: 특정 폴더에서 바로 열기

윈도우 탐색기 주소창에 cmd를 입력하고 엔터를 치면, 현재 보고 있는 폴더 경로에서 바로 터미널이 열립니다. 그 상태에서 gemini를 입력하면 해당 폴더의 파일들을 제가 바로 인식할 수 있습니다!

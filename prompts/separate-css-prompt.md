# 스타일 외부 파일 분리 프롬프트

## 요청 내용
`[HTML 파일 경로]`의 style을 외부 css로 수정해줘.

## 적용 방법
1. HTML 파일 내의 `<style>` 태그 내용을 추출하여 `css/` 폴더 내에 해당 파일명과 매칭되는 `.css` 파일을 생성합니다.
2. CSS 파일 내에서 배경 이미지(`url()`) 등의 상대 경로가 있다면, CSS 파일의 위치에 맞춰 경로를 수정합니다 (예: `./img/` -> `../img/`).
3. HTML 파일에서 `<style>` 태그를 삭제하고, 생성한 외부 CSS 파일을 연결하는 `<link>` 태그를 `<head>` 내에 추가합니다.

## 참고 사항
- 프로젝트의 기존 디렉토리 구조(예: `02-css/css/`)를 존중하여 파일을 생성합니다.
- HTML 파일과 CSS 파일 간의 올바른 연결을 위해 `href` 경로를 정확히 지정해야 합니다.

현재 깃허브와의 실제 인증 연동(SSH 또는 CLI 로그인)은 완료되지 않은 상태입니다. 깃허브를 연동하여 사용하시려면 아래 방법 중 하나를 진행해 주세요.

방법 A: HTTPS 로그인 연동 (가장 간편한 방법)
1. 터미널에서 현재 폴더를 Git 저장소로 초기화합니다:
   git init
2. 본인의 깃허브 원격 저장소를 연결합니다:
   git remote add origin <깃허브 저장소 주소>
3. 첫 push를 시도하면 자격 증명(Git Credential Manager) 창이 뜨며 웹 브라우저 로그인을 통해 자동으로 연동됩니다:
   git push -u origin main

방법 B: SSH 키 등록
1. SSH 키 쌍을 생성한 후, 공개키(id_ed25519.pub 등)를 깃허브 계정의 [Settings -> SSH and GPG keys]에 등록해야 합니다.
